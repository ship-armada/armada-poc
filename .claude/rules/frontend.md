# Frontend Rules

**The frontend (`usdc-v2-frontend/`) is a temporary stopgap for testing.** It will be replaced with a purpose-built application.

- Residual Namada/Noble/Cosmos code paths exist and can be left alone.
- Do not invest significant effort in frontend architecture improvements.
- Bug fixes and feature additions to support testing flows are fine.
- The frontend requires: Anvil chains running, contracts deployed, and the Armada relayer running.
